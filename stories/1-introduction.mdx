# @holochain-open-dev/peer-status

Zome and UI module to display the online/offline status of agents in a holochain app.

Existing functionalities:

- Ping an agent to check whether they are online or not.

This module follows the [holochain-open-dev](https://github.com/holochain-open-dev/) pattern of developing holochain modules. Read [its documentation](https://holochain-open-dev.github.io) to understand more about its motivation and the big picture.

To integrate this module into your application visit [Setting up the Backend](?path=/docs/backend-setting-up-the-zomes--docs) and [Setting up the Frontend](?path=/docs/frontend-setting-up-the-frontend--docs).
